Research

Loosely-coupled In-situ Analysis on GPU-accelerated Platforms

Portable Large-scale Image-analysis on Hetrogeneous HPC Platforms

  • Performance-based large-scale medical image analysis workflows deployment on federated high-performance resources. (MICCAI-HPC '15)
  • HetroCV: auto-tuning framework and runtime for image-processing applications on hetrogeneous platforms. Proposed and implemented HetroCV auto-tuning framework and runtime for image processing application on heterogeneous CPU- MIC platforms. (ICPP-P2S2 '15)

Computer-aided Analysis for Prostate Histopathology Images

  • Proposed and implemented glandular structure feature based CAD algorithm for prostate histopathology images Gleason grading. (EMBC '15).
  • Proposed and implemented global color histogram-texture feature based CAD algorithm for prostate histopathology images Gleason grading. (MICCAI-HPC '14).

Large-scale Content-based Image Retrieval

  • Proposed and implemented the GPGPU-based content-based image retrieval algorithm and a 2-level (image/patch) load balancing scheme. (MICCAI-HPC '13)
  • Proposed and implemented the relevance-feedback algorithm for content-based image retrieval results refining; tested large-scale CBIR algorithm on CometCloud cloud computing platform. (BMC Bioinformatics 2014)
